                                                                                  ABSTRACT

The purpose of the project entitled as “Hospital Administration-TRUST HIS” is to computerize the front office management of the hospital to develop software which is user friendly, simple, fast and cost effective.TRUST HIS furnishes desirable care and in cooperates clinical work flows which are tailed to your needs.This customizable hospital administration system  is an integrated health care solution  which include outpatient and inpatient overview, ward overview, online appointment overview, medical electronic billing, accounting, etc.The software has the facility to give a unique id for every patient and stores the details of each patient and the staff automatically.It includes a search facility to know the current status of each room.User can search availability of a doctor and the details of a patient using the patient-id.The Hospital Admin can be entered using a username and password.It is accessible either by an administrator or receptionist. Only they can add data into the database.The data can be retrieved easily.Traditionally, it was done manually.It is a totally web based application built using the very latest user interface technologies like Bootstrap ,Javascript, CSS, HTML, Django Framework, python programming language to provide an enormous user friendly experience.
